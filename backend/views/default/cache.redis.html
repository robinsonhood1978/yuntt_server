{$lang.operation_tip}

  • 1. Redis是基于内存的，作为缓存使用可以有效提高页面访问速度
  • 2. 在开启Redis缓存之前，请确保在服务端已经安装了Redis服务否则会造成页面无法访问，请开放6379端口（默认）及设置好白名单
  • 3. Redis服务可以为本地安装（使用宝塔可以一键安装），也可以是云端，例如：阿里云的【云数据库Redis】产品，开箱即用
  • 4. !!!如果配置后导致网站无法访问，请打开文件根目录/frontend/web/data/config.php 删除节点【cache】
